# RH0832 - Struct attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0832 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Struct attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[First, Second]
internal struct Example { }
```

### Correction

```cs
[First]
[Second]
internal struct Example { }
```
