# RH0341 — Opening square brackets must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0341       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No space may appear immediately after an opening square bracket `[`.

## Why is this a problem?

A leading space inside brackets is non-standard and makes indexer and collection expressions harder to scan.

## How to fix it

Remove the space immediately after the opening square bracket. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = [0];
        _ = values[ 0];
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = [0];
        _ = values[0];
    }
}
```
