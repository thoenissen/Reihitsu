# RH0843 - Event attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0843 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Event attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [First] internal event EventHandler? Changed;
}
```

### Correction

```cs
internal class Example
{
    [First]
    internal event EventHandler? Changed;
}
```
