# RH5403 — Statement must not be on a single line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH5403       |
| **Category** | Layout |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

A block body that contains one or more statements must not be written on a single line. Each statement must appear on its own line.

## Why is this a problem?

Collapsing a block onto one line hides its content and makes it difficult to read, step through in a debugger, or set a breakpoint on an individual statement.

## How to fix it

Split the block so each statement is on its own line. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        if (true) { return; }
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
            return;
        }
    }
}
```
