# RH0840 - Property attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0840 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Property attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [First, Second]
    internal int Value { get; set; }
}
```

### Correction

```cs
internal class Example
{
    [First]
    [Second]
    internal int Value { get; set; }
}
```
