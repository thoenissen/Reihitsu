# RH0380 — Parameter list must follow declaration

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0380       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that parameter list must follow declaration.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method
    (int value)
    {
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method(int value)
    {
    }
}
```
