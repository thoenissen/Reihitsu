# RH0825 - Assembly attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0825 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Assembly attributes must be placed on their own line.

## Examples

### Violation

```cs
[assembly: First] internal class Example { }
```

### Correction

```cs
[assembly: First]
internal class Example { }
```
