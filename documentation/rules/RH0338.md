# RH0338 — Preprocessor keywords must not be preceded by space

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0338       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that preprocessor keywords must not be preceded by space.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
    #pragma warning disable CS0168
internal class TestClass
{
}
```

### Correction

```cs
#pragma warning disable CS0168
internal class TestClass
{
}
```
