# RH5520 — Event attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5520 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Event attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [First, Second]
    internal event EventHandler? Changed;
}
```

### Correction

```cs
internal class Example
{
    [First]
    [Second]
    internal event EventHandler? Changed;
}
```
