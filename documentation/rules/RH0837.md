# RH0837 - Method attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0837 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Method attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [First] internal void Execute() { }
}
```

### Correction

```cs
internal class Example
{
    [First]
    internal void Execute() { }
}
```
