# RH6003 — Semicolons must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6003       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

A semicolon must not be preceded by a space.

## Why is this a problem?

A space before a semicolon is non-standard and creates visual noise that draws attention away from the statement content.

## How to fix it

Remove any space immediately before the semicolon. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        var value = 0 ;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        var value = 0;
    }
}
```
