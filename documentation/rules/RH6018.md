# RH6018 — Negative signs must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6018       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

The unary minus operator used as a sign must not be followed by a space.

## Why is this a problem?

A space after a unary minus makes it ambiguous whether the operator is unary or binary, reducing readability.

## How to fix it

Remove the space immediately after the unary minus sign. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int value = - 1;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int value = -1;
    }
}
```
