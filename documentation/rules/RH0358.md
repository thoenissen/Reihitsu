# RH0358 — Code must not contain trailing whitespace

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0358       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule checks whether a line ends with spaces or tabs before the line break. Trailing whitespace creates noisy diffs and makes formatting less predictable.

## Why is this a problem?

Trailing whitespace does not affect execution, but it increases visual noise and causes avoidable diff churn.

## How to fix it

Remove the trailing whitespace at the end of the line.
