# RH0842 - Field attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0842 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Field attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [First, Second]
    private int _value;
}
```

### Correction

```cs
internal class Example
{
    [First]
    [Second]
    private int _value;
}
```
