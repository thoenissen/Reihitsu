# RH0351 — Increment/decrement symbols must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0351       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that increment/decrement symbols must be spaced correctly.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int value = 0;
        value ++;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int value = 0;
        value++;
    }
}
```
