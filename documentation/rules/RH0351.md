# RH0351 — Increment/decrement symbols must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0351       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

The increment `++` and decrement `--` operators must not be separated from their operand by a space.

## Why is this a problem?

A space between the operator and its operand makes the expression look like two separate tokens and reduces readability.

## How to fix it

Remove the space between the operand and the `++` or `--` operator. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int value = 0;
        value ++;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int value = 0;
        value++;
    }
}
```
