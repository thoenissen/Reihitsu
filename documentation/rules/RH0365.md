# RH0365 — Code must not contain multiple blank lines in a row

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0365       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that code must not contain multiple blank lines in a row.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int first = 0;


        int second = 1;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int first = 0;

        int second = 1;
    }
}
```
