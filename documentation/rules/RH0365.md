# RH0365 — Code must not contain multiple blank lines in a row

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0365       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

At most one blank line may appear consecutively anywhere in a source file.

## Why is this a problem?

Multiple consecutive blank lines add unnecessary vertical space and push code off-screen, making it harder to read and review.

## How to fix it

Reduce any sequence of two or more blank lines to a single blank line. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int first = 0;


        int second = 1;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int first = 0;

        int second = 1;
    }
}
```
