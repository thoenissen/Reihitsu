# RH0367 — Opening brace must not be preceded by blank line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0367       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No blank line may appear between a declaration and its opening brace.

## Why is this a problem?

A blank line before the opening brace separates a declaration from its body, making it harder to see that the brace belongs to that declaration.

## How to fix it

Remove the blank line between the declaration and its opening brace. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()

    {
        int value = 0;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int value = 0;
    }
}
```
