# RH0432 - <value> tag must not be used

| Property | Value |
|----------|-------|
| **ID** | RH0432 |
| **Category** | Documentation |
| **Severity** | Warning |
| **Code Fix** | ❌ |

## Description

This rule forbids direct use of the &lt;value&gt; tag in XML documentation.

## Why is this a problem?

The &lt;value&gt; tag duplicates property information without adding useful structure. A clear &lt;summary&gt; is easier to read and keeps property documentation consistent.

## How to fix it

Move the important information into the property summary and remove the &lt;value&gt; tag.

## Examples

### Violation

```cs
/// <summary>Gets the current count</summary>
/// <value>The current count</value>
internal int Count { get; }
```

### Correction

```cs
/// <summary>
/// Gets the current count
/// </summary>
internal int Count { get; }
```
