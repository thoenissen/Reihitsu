# RH0373 — Braces must not be omitted from multi-line child statements

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0373       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that braces must not be omitted from multi-line child statements.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
            Other(1);
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
            Other(1);
        }
    }
}
```
