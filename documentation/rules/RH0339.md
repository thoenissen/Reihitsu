# RH0339 — Operator keyword must be followed by space

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0339       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that operator keyword must be followed by space.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    public static TestClass operator+(TestClass left, TestClass right) => left;
}
```

### Correction

```cs
internal class TestClass
{
    public static TestClass operator +(TestClass left, TestClass right) => left;
}
```
