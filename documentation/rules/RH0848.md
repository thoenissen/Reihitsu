# RH0848 - Parameter attributes must use merged attribute lists

| Property | Value |
|----------|-------|
| **ID** | RH0848 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Parameter attributes must use one merged attribute list.

## Examples

### Violation

```cs
internal class Example
{
    internal void Execute([First][Second] int value)
    {
    }
}
```

### Correction

```cs
internal class Example
{
    internal void Execute([First, Second] int value)
    {
    }
}
```
