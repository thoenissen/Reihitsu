# RH0355 — Colons must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0355       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that colons must be spaced correctly.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass:System.IDisposable
{
    public void Dispose()
    {
    }
}
```

### Correction

```cs
internal class TestClass : System.IDisposable
{
    public void Dispose()
    {
    }
}
```
