# RH5503 — Module attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5503 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Module attributes must be placed on their own line.

## Examples

### Violation

```cs
[module: First] internal class Example { }
```

### Correction

```cs
[module: First]
internal class Example { }
```
