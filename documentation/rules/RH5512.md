# RH5512 — Constructor attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5512 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Constructor attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [First, Second]
    internal Example() { }
}
```

### Correction

```cs
internal class Example
{
    [First]
    [Second]
    internal Example() { }
}
```
