# RH0345 — Opening generic brackets must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0345       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No space may appear immediately before the opening angle bracket `<` of a generic type argument list.

## Why is this a problem?

A space before the opening angle bracket separates the type name from its type arguments, making generic types look different from the rest of the language and harder to recognize at a glance.

## How to fix it

Remove the space immediately before the opening generic angle bracket. The code fix applies this change automatically.

## Examples

### Violation

```cs
using System.Collections.Generic;
internal class TestClass
{
    void Method()
    {
        _ = new List <int>();
    }
}
```

### Correction

```cs
using System.Collections.Generic;
internal class TestClass
{
    void Method()
    {
        _ = new List<int>();
    }
}
```
