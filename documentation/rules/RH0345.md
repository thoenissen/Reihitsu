# RH0345 — Opening generic brackets must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0345       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that opening generic brackets must be spaced correctly.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
using System.Collections.Generic;
internal class TestClass
{
    void Method()
    {
        _ = new List <int>();
    }
}
```

### Correction

```cs
using System.Collections.Generic;
internal class TestClass
{
    void Method()
    {
        _ = new List<int>();
    }
}
```
