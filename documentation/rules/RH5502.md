# RH5502 — Assembly attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5502 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Assembly attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[assembly: First, Second]
internal class Example { }
```

### Correction

```cs
[assembly: First]
[assembly: Second]
internal class Example { }
```
