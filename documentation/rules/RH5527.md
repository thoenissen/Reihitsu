# RH5527 — Return value attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5527 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Return value attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [return: First] internal int Execute() => 0;
}
```

### Correction

```cs
internal class Example
{
    [return: First]
    internal int Execute() => 0;
}
```
