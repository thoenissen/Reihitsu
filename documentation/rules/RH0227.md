# RH0227 — The given namespace is not allowed

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0227       |
| **Category** | Naming       |
| **Severity** | Warning      |
| **Code Fix** | ✗            |

## Description

Checks namespace declarations against a list of disallowed patterns defined in the `reihitsu.json` configuration file. This rule flags any namespace that matches a forbidden pattern.

## Why is this a problem?

Some namespaces may be forbidden by project conventions, architectural boundaries, or organizational policies. Allowing disallowed namespaces can lead to coupling violations or naming conflicts.

## How to fix it

Rename the namespace to a value that is allowed by the project configuration.

## Examples

### Violation

```cs
// Where "ForbiddenNamespace" is disallowed in reihitsu.json
namespace ForbiddenNamespace
{
}
```

### Correction

```cs
namespace AllowedNamespace
{
}
```
