# RH3107 — Unnecessary attribute constructor parentheses should be removed

| Property | Value |
|----------|-------|
| **ID** | RH3107 |
| **Category** | Clarity |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

Attributes without constructor arguments should not keep an empty argument list.

## Why is this a problem?

`[Obsolete()]` is noisier than `[Obsolete]` and conveys no additional information.

## How to fix it

Remove the empty parentheses from the attribute usage.

## Examples

### Violation

```cs
[System.Obsolete()]
internal class TestClass { }
```

### Correction

```cs
[System.Obsolete]
internal class TestClass { }
```
