# RH0833 - Enum attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0833 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Enum attributes must be on their own line.

## Examples

### Violation

```cs
[First] internal enum Example { A }
```

### Correction

```cs
[First]
internal enum Example { A }
```
