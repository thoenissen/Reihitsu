# RH0847 - Parameter attributes must stay inline

| Property | Value |
|----------|-------|
| **ID** | RH0847 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Parameter attributes must stay inline with the parameter declaration.

## Examples

### Violation

```cs
internal class Example
{
    internal void Execute([First]
                          int value)
    {
    }
}
```

### Correction

```cs
internal class Example
{
    internal void Execute([First] int value)
    {
    }
}
```
