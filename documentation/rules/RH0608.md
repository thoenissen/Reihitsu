# RH0608 — Using directives must be ordered alphabetically by namespace

| Property | Value |
|----------|-------|
| **ID** | RH0608 |
| **Category** | Ordering |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

This rule requires regular `using` directives to be sorted alphabetically by namespace within their group.

## Why is this a problem?

Alphabetical ordering makes it easier to find imports quickly and keeps file headers consistent.

## How to fix it

Sort the regular `using` directives by namespace name.

## Examples

### Violation

```cs
using System.Linq;
using System.Collections.Generic;
```

### Correction

```cs
using System.Collections.Generic;
using System.Linq;
```
