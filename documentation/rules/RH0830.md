# RH0830 - Class attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0830 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Class attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[First, Second]
internal class Example { }
```

### Correction

```cs
[First]
[Second]
internal class Example { }
```
