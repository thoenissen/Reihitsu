# RH5529 — Generic parameter attributes must use merged attribute lists

| Property | Value |
|----------|-------|
| **ID** | RH5529 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Generic parameter attributes must use one merged attribute list.

## Examples

### Violation

```cs
internal class Example<[typevar: First][typevar: Second] T>
{
}
```

### Correction

```cs
internal class Example<[typevar: First, Second] T>
{
}
```
