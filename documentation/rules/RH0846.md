# RH0846 - Interface attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0846 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Interface attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[First, Second]
internal interface IExample { }
```

### Correction

```cs
[First]
[Second]
internal interface IExample { }
```
