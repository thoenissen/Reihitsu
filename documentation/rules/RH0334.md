# RH0334 — Keywords must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0334       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

A space is required between a keyword (such as `if`, `for`, or `while`) and the opening parenthesis that follows it.

## Why is this a problem?

Missing spaces around keywords make control-flow constructs look like method calls and reduce readability.

## How to fix it

Insert a space between the keyword and the opening parenthesis. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        if(true)
        {
        }
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
        }
    }
}
```
