# RH6008 — Closing square brackets must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6008       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No space may appear immediately before a closing square bracket `]`.

## Why is this a problem?

A trailing space inside brackets is non-standard and makes indexer and collection expressions harder to scan.

## How to fix it

Remove the space immediately before the closing square bracket. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = [0];
        _ = values[0 ];
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = [0];
        _ = values[0];
    }
}
```
