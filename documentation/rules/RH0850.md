# RH0850 - Delegate attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0850 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Delegate attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[First, Second]
internal delegate void ExampleDelegate();
```

### Correction

```cs
[First]
[Second]
internal delegate void ExampleDelegate();
```
