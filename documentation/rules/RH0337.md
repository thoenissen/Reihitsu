# RH0337 — Documentation lines must begin with single space

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0337       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that documentation lines must begin with single space.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    ///Summary
    void Method()
    {
    }
}
```

### Correction

```cs
internal class TestClass
{
    /// Summary
    void Method()
    {
    }
}
```
