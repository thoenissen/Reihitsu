# RH0839 - Property attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0839 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Property attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [First] internal int Value { get; set; }
}
```

### Correction

```cs
internal class Example
{
    [First]
    internal int Value { get; set; }
}
```
