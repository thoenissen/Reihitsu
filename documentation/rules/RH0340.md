# RH0340 — Opening parenthesis must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0340       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that opening parenthesis must be spaced correctly.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    int Method()
    {
        return ( 0);
    }
}
```

### Correction

```cs
internal class TestClass
{
    int Method()
    {
        return (0);
    }
}
```
