# RH5507 — Struct attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5507 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Struct attributes must be on their own line.

## Examples

### Violation

```cs
[First] internal struct Example { }
```

### Correction

```cs
[First]
internal struct Example { }
```
