# RH0205 — Enum member names should be in PascalCase

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0205       |
| **Category** | Naming       |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

Checks that enum member names follow PascalCase naming conventions. Each word in the name should start with an uppercase letter, with no underscores or other separators.

## Why is this a problem?

PascalCase for enum members is a standard C# naming convention. Inconsistent casing makes the code harder to read and breaks expectations for consumers of the API.

## How to fix it

Rename the enum members to use PascalCase.

## Examples

### Violation

```cs
public enum Color
{
    red,
    green
}
```

### Correction

```cs
public enum Color
{
    Red,
    Green
}
```
