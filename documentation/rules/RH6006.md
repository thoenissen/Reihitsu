# RH6006 — Opening parenthesis must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6006       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No space may appear immediately after an opening parenthesis.

## Why is this a problem?

A leading space inside parentheses is non-standard and draws the eye inward, making it harder to identify the start of the enclosed expression.

## How to fix it

Remove the space immediately after the opening parenthesis. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    int Method()
    {
        return ( 0);
    }
}
```

### Correction

```cs
internal class TestClass
{
    int Method()
    {
        return (0);
    }
}
```
