# RH0308 — foreach-Statement should be preceded by a blank line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0308       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✗            |

## Description

This rule checks that `foreach` statements are preceded by a blank line. The rule does not apply when the `foreach` statement is the first statement in a block.

## Why is this a problem?

Separating statements with blank lines improves code readability by visually grouping related logic. A blank line before a `foreach` loop makes it clear where iteration begins and separates it from preceding setup code.

## How to fix it

Add a blank line before the `foreach` statement. The rule does not apply when the statement is the first statement in a block.

## Examples

### Violation

```cs
var items = GetItems();
foreach (var item in items)
{
    Process(item);
}
```

### Correction

```cs
var items = GetItems();

foreach (var item in items)
{
    Process(item);
}
```
