# RH0835 - Constructor attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0835 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Constructor attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [First] internal Example() { }
}
```

### Correction

```cs
internal class Example
{
    [First]
    internal Example() { }
}
```
