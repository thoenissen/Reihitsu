# RH0359 — Braces for multi-line statements must not share line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0359       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that braces for multi-line statements must not share line.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        if (true) {
            int value = 0;
        }
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
            int value = 0;
        }
    }
}
```
