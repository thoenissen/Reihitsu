# RH5103 — Code must not contain multiple statements on one line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH5103       |
| **Category** | Layout |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule checks whether sibling statements in a block or switch section start on the same line.

## Why is this a problem?

Placing multiple statements on one line reduces readability and makes changes harder to review.

## How to fix it

Move each statement to its own line.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int first = 1; int second = 2;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int first = 1;
        int second = 2;
    }
}
```
