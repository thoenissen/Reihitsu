# RH7001 — File may only contain a single namespace

| Property | Value |
|----------|-------|
| **ID** | RH7001 |
| **Category** | Organization |
| **Severity** | Warning |
| **Code Fix** | ❌ |

## Description

A source file should contain at most one top-level namespace declaration.

## Why is this a problem?

Multiple top-level namespaces in one file make the file harder to navigate and blur organizational boundaries.

## How to fix it

Move the additional namespace declarations into separate files.

## Examples

### Violation

```cs
namespace First
{
    internal class A { }
}

namespace Second
{
    internal class B { }
}
```

### Correction

*File `A.cs`:*

```cs
namespace First
{
    internal class A { }
}
```

*File `B.cs`:*

```cs
namespace Second
{
    internal class B { }
}
```
