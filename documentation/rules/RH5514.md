# RH5514 — Method attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5514 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Method attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [First, Second]
    internal void Execute() { }
}
```

### Correction

```cs
internal class Example
{
    [First]
    [Second]
    internal void Execute() { }
}
```
