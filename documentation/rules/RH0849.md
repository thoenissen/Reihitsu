# RH0849 - Delegate attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH0849 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Delegate attributes must be on their own line.

## Examples

### Violation

```cs
[First] internal delegate void ExampleDelegate();
```

### Correction

```cs
[First]
internal delegate void ExampleDelegate();
```
