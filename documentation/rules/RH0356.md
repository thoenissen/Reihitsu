# RH0356 — No space after new for implicitly typed arrays

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0356       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

There must be no space between the `new` keyword and the opening bracket `[` of an implicitly typed array creation expression.

## Why is this a problem?

A space between `new` and `[]` is inconsistent with the standard style for array creation and makes it look like two separate tokens.

## How to fix it

Remove the space between `new` and `[`. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = new [] { 1 };
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        int[] values = new[] { 1 };
    }
}
```
