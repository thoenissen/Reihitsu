# RH6021 — Colons must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6021       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

A colon `:` used in a base-type list, case label, or ternary expression must be surrounded by exactly one space on each side.

## Why is this a problem?

A colon without surrounding spaces visually fuses the tokens on either side, making inheritance declarations and case labels harder to read.

## How to fix it

Add a space before and after the colon. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass:System.IDisposable
{
    public void Dispose()
    {
    }
}
```

### Correction

```cs
internal class TestClass : System.IDisposable
{
    public void Dispose()
    {
    }
}
```
