# RH0824 — Indexer bracketed arguments should be single lined

| Property | Value |
|----------|-------|
| **ID** | RH0824 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

This rule requires indexer argument lists (`target[...]`) to stay on a single line.

## Why is this a problem?

Multiline indexer argument lists reduce readability and make simple element access harder to parse while scanning expressions.

## How to fix it

Collapse indexer arguments so that `[` and `]` and all arguments are on the same line.

The code fix reformats the surrounding code automatically using the formatter.

## Examples

### Violation

```cs
var value = matrix[
    1,
    2];
```

### Correction

```cs
var value = matrix[1, 2];
```
