# RH6013 — Opening attribute brackets must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH6013       |
| **Category** | Spacing |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

No space may appear immediately after the opening bracket `[` of an attribute.

## Why is this a problem?

A leading space inside an attribute's brackets is non-standard and makes the attribute harder to recognize as a compact annotation.

## How to fix it

Remove the space immediately after the opening attribute bracket. The code fix applies this change automatically.

## Examples

### Violation

```cs
[ System.Obsolete]
internal class TestClass
{
}
```

### Correction

```cs
[System.Obsolete]
internal class TestClass
{
}
```
