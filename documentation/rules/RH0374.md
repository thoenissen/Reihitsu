# RH0374 — Use braces consistently

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0374       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that use braces consistently.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
            return;
        }
        else
            return;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        if (true)
        {
            return;
        }
        else
        {
            return;
        }
    }
}
```
