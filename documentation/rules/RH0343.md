# RH0343 — Opening braces must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0343       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

An opening brace `{` must be preceded by exactly one space when it follows a non-whitespace character on the same line.

## Why is this a problem?

An opening brace that is not separated by a space visually merges with the preceding token, making it harder to identify the start of the block.

## How to fix it

Insert a single space before the opening brace. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    int Property{ get; set; }
}
```

### Correction

```cs
internal class TestClass
{
    int Property { get; set; }
}
```
