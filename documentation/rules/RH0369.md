# RH0369 — While-do footer must not be preceded by blank line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0369       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that while-do footer must not be preceded by blank line.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        do
        {
        }

        while (true);
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        do
        {
        }
        while (true);
    }
}
```
