# RH0603 — Partial elements must declare access modifier

| Property | Value |
|----------|-------|
| **ID** | RH0603 |
| **Category** | Ordering |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

This rule requires partial type declarations to state their accessibility explicitly.

## Why is this a problem?

Partial types are split across files, so relying on the default accessibility makes the intended visibility harder to see. Being explicit avoids guesswork when reading only one part of the type.

## How to fix it

Add the correct access modifier, such as `internal`, `public`, `private protected`, or `protected internal`.

## Examples

### Violation

```cs
partial class Example
{
}
```

### Correction

```cs
internal partial class Example
{
}
```
