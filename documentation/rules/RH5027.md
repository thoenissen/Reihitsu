# RH5027 — While-do footer must not be preceded by blank line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH5027       |
| **Category** | Layout |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

The `while` clause of a `do`-`while` statement must appear on the line immediately following the closing brace of the `do` block, without a blank line in between.

## Why is this a problem?

A blank line between the closing brace of a `do` block and its `while` clause breaks the visual pairing of the two parts of the statement.

## How to fix it

Remove the blank line between the closing brace of the `do` block and the `while` clause. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        do
        {
        }

        while (true);
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        do
        {
        }
        while (true);
    }
}
```
