# RH0361 — Element must not be on a single line

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0361       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that element must not be on a single line.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass { }
```

### Correction

```cs
internal class TestClass
{
}
```
