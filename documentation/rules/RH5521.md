# RH5521 — Interface attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5521 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Interface attributes must be on their own line.

## Examples

### Violation

```cs
[First] internal interface IExample { }
```

### Correction

```cs
[First]
internal interface IExample { }
```
