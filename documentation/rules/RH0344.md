# RH0344 — Closing braces must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0344       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

A closing brace `}` must be followed by exactly one space when it is followed by a non-whitespace character on the same line.

## Why is this a problem?

A closing brace that is not separated by a space from the following token visually merges the end of the block with subsequent code, reducing readability.

## How to fix it

Insert a single space after the closing brace. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    int Property { get; set;}
}
```

### Correction

```cs
internal class TestClass
{
    int Property { get; set; }
}
```
