# RH5505 — Class attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5505 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Class attributes must be on their own line.

## Examples

### Violation

```cs
[First] internal class Example { }
```

### Correction

```cs
[First]
internal class Example { }
```
