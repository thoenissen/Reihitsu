# RH0828 - Module attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0828 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Module attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[module: First, Second]
internal class Example { }
```

### Correction

```cs
[module: First]
[module: Second]
internal class Example { }
```
