# RH0349 — Nullable type symbols must not be preceded by space

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0349       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that nullable type symbols must not be preceded by space.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method(int ? value)
    {
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method(int? value)
    {
    }
}
```
