# RH0349 — Nullable type symbols must not be preceded by space

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0349       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

The nullable type operator `?` must not be preceded by a space.

## Why is this a problem?

A space before `?` visually separates the operator from the base type, making nullable type annotations harder to read.

## How to fix it

Remove the space immediately before the `?` operator. The code fix applies this change automatically.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method(int ? value)
    {
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method(int? value)
    {
    }
}
```
