# RH0371 — Code must not contain blank lines at the start of the file

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0371       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule checks whether a file starts with one or more blank lines before the first code or comment line.

## Why is this a problem?

Leading blank lines add inconsistent vertical padding and make file structure less predictable.

## How to fix it

Remove the blank lines at the start of the file so the content begins on the first line.

## Examples

### Violation

```cs

internal class TestClass
{
}
```

### Correction

```cs
internal class TestClass
{
}
```
