# RH0852 - Return value attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH0852 |
| **Category** | Formatting |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Return value attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
internal class Example
{
    [return: First, Second]
    internal int Execute() => 0;
}
```

### Correction

```cs
internal class Example
{
    [return: First]
    [return: Second]
    internal int Execute() => 0;
}
```
