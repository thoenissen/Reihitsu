# RH0110 — Unnecessary delegate parentheses should be removed

| Property | Value |
|----------|-------|
| **ID** | RH0110 |
| **Category** | Design |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

Anonymous methods without parameters should not keep an empty parameter list.

## Why is this a problem?

`delegate() { }` is noisier than necessary when there are no parameters to describe.

## How to fix it

Remove the empty parentheses so the anonymous method uses `delegate { }`.
