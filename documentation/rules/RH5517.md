# RH5517 — Field attributes must be on their own line

| Property | Value |
|----------|-------|
| **ID** | RH5517 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Field attributes must be on their own line.

## Examples

### Violation

```cs
internal class Example
{
    [First] private int _value;
}
```

### Correction

```cs
internal class Example
{
    [First]
    private int _value;
}
```
