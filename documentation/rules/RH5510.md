# RH5510 — Enum attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5510 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Enum attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[First, Second]
internal enum Example { A }
```

### Correction

```cs
[First]
[Second]
internal enum Example { A }
```
