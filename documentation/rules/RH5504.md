# RH5504 — Module attribute lists must use one attribute per list

| Property | Value |
|----------|-------|
| **ID** | RH5504 |
| **Category** | Layout |
| **Severity** | Warning |
| **Code Fix** | Yes |

## Description

Module attribute lists must contain exactly one attribute.

## Examples

### Violation

```cs
[module: First, Second]
internal class Example { }
```

### Correction

```cs
[module: First]
[module: Second]
internal class Example { }
```
