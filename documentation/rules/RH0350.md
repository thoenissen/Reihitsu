# RH0350 — Member access symbols must be spaced correctly

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0350       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

This rule ensures that member access symbols must be spaced correctly.

## Why is this a problem?

Inconsistent formatting makes code harder to read and creates unnecessary noise in diffs.

## How to fix it

Adjust the affected formatting manually or apply the provided code fix.

## Examples

### Violation

```cs
internal class TestClass
{
    void Method()
    {
        _ = string .Empty;
    }
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
        _ = string.Empty;
    }
}
```
