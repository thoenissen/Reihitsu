# RH0614 — Using static directives must be ordered alphabetically

| Property | Value |
|----------|-------|
| **ID** | RH0614 |
| **Category** | Ordering |
| **Severity** | Warning |
| **Code Fix** | ✓ |

## Description

This rule requires `using static` directives to be sorted alphabetically by type name.

## Why is this a problem?

Alphabetical ordering makes static import blocks easier to scan and keeps them consistent with the rest of the using section.

## How to fix it

Sort `using static` directives alphabetically.

## Examples

### Violation

```cs
using static System.Math;
using static System.Console;
```

### Correction

```cs
using static System.Console;
using static System.Math;
```
