# RH0357 — Tab characters must not be used

| Property     | Value        |
|--------------|--------------|
| **ID**       | RH0357       |
| **Category** | Formatting   |
| **Severity** | Warning      |
| **Code Fix** | ✓            |

## Description

Tab characters must not appear in source code outside of string, character, and interpolated-string literals. All indentation and horizontal spacing must use spaces.

## Why is this a problem?

Tabs render differently depending on the editor or tool. Mixing tabs and spaces causes inconsistent indentation across environments and is a common source of diff noise.

## How to fix it

Replace each tab character with the appropriate number of spaces. The code fix substitutes the tab automatically.

## Examples

### Violation

```cs
internal class TestClass
{
	void Method()
	{
	}
}
```

### Correction

```cs
internal class TestClass
{
    void Method()
    {
    }
}
```
